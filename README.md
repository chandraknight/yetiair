# FastAPI Project

This is a FastAPI project with a clean architecture structure.

## Setup

1.  Create a virtual environment:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Run the application:
    ```bash
    python run.py
    ```

## Structure

The project follows a modular structure separated by concerns (handlers, services, repositories, models).
